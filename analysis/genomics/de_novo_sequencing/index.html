De Novo Sequencing analysis

In this page we list the set of algorithms currently implemented at the i3S bioinformatics platform. Due to the high variablility between de novo assemblers, a conjugation of the following workflows is recommended in order to optimize contig length (e.g. N50), genome coverage and maximum, median and average contig size.
