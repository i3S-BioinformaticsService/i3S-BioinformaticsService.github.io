Resequencing

In this page we list the set of algorithms currently implemented at the i3S bioinformatics platform. Selection of the alignment software will take in consideration the sequencing method employed, i.e. short or long read sequencing. The provided short-read alignment workflows have similar alignment efficiencies for local read mapping. For global read mapping, bowtie2 is recommended since BWA lacks this capability.
