    • Cell type identification

    A key step in any scRNA-seq is the annotation of cell clusters with biological meaningful cell types. This process traditionally involves an automatic clustering step followed by annotation of individual cells or cell clusters using a set of marker genes specifically expressed in a known cell type of use existing annotated single-cell maps as reference. Although, the clustering an annotation step can be performed in an automated fashion, often a manual curation step is required for lower confidence and conflicting or absent cell labels. This manual curation step, when required, will be left to the user.
