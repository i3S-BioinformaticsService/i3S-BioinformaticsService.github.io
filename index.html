    About this page:

    This website aims to provide the bioinformatic workflows currently in use at the i3S bioinformatics service. For reproducibility purposes all workflows are based on docker images and linked between tools using the common workflow language (CWL). Please click on the image containing the analysis you want to perform for more information. For custom analyis, please contact the i3S bioinformatics service.
